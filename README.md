# Ad Astra: Mythic Metals

**Ad Astra: Mythic Metals** adds new planetary ore variants that integrate seamlessly with both [Ad Astra](https://modrinth.com/mod/ad-astra) and [Mythic Metals](https://modrinth.com/mod/mythicmetals).

---

## Features

- **4 New Planetary Ores**
    - **Mars Carmot Ore** — Found on Mars
    - **Moon Stormyx Ore** — Found on  the Moon
    - **Mercury Palladium Ore** — Found on Mercury
    - **Venus Orichalcum Ore** — Found on Venus
    - **Glacio Unobtainium Ore** — Found on Glacio
  
**These no longer spawn in the Overworld or Nether**

---

## Installation

1. Install [Fabric Loader](https://fabricmc.net/) for Minecraft 1.20.1
2. Download and install the required dependencies listed above
3. Download **Ad Astra: Mythic Metals**
4. Place the mod `.jar` file in your `mods` folder
